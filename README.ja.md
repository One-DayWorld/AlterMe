## ✨ なぜ AlterMe？

多くの AI チャットアプリは、誰かのサーバーに薄い皮をかぶせただけ —— あなたのメッセージと鍵は**相手の**バックエンドを通ります。**AlterMe にはバックエンドが一切ありません。** 純粋な端末内 WebView が、**あなたの**鍵でモデル API に直接話しかけます。あなたについて覚えたことは、すべて端末の中に残ります。

しかも小さくて改造しやすい：JS ファイルは約 6 個、フレームレスでビルド不要。10 分で「頭脳」全体を読み切れます。

## 🎯 主な機能

- 🎭 **複数の切り替え可能なキャラ** —— 人格は 1 つだけではありません：パートナー / 聞き役 / 遊び相手 / 理性派などの内蔵テンプレを自由に追加・編集・削除。各キャラが独自の人格・専用ルール・会話履歴・絆レベルを持ち、上部バーからワンタップで切り替え、画像アイコンも設定可能。「どのモデル？」と聞けば正直に答えます(GPT のフリはしません)。
- 🧠 **本物の記憶(全キャラで共有)** —— 会話からあなたの長期「プロフィール」をそっと抽出し、すべてのキャラがこの「あなた理解」を共有します。さらに**記事を食べさせる**(リンクや本文を貼る)ことで、興味の理解を形づくれます。
- 💞 **絆とレベル(キャラごとに独立)** —— 各キャラと個別に経験値が貯まり、それぞれレベルアップ。小さな仕掛けですが、不思議と戻りたくなります。
- 🔀 **2 つのモデルをワンタップ** —— Qwen-Plus(Alibaba)と DeepSeek、どちらも OpenAI 互換。いつでも切り替え可能。
- 🔒 **設計レベルのプライバシー** —— API キーは端末内のみ保存。こちらのサーバーには何も送りません(そもそも存在しません)。記憶はワンボタンで全消去。
- 📱 **Google が届かない場所でも動く** —— 純 WebView で GMS 依存ゼロ。Huawei / HarmonyOS 4.2 にも通常の APK としてそのまま入ります。
- 🪶 **改造前提の設計** —— 素の HTML/CSS/JS、ツールチェーン不要。`www/` を開けばすぐ編集できます。

## 🚀 クイックスタート

### まずブラウザで試す(開発時)

`www/` はただの静的サイトです：

```bash
cd www
python3 -m http.server 8080
# http://localhost:8080 を開く
```

> デスクトップブラウザではモデルへの直接リクエストが CORS に当たることがありますが、これは開発時のみ。端末では CapacitorHttp 経由になり問題なく動きます。

### APK をビルド(Android / HarmonyOS)

**Node + Android Studio + Android SDK** が必要です。

```bash
npm install
npx cap add android      # 初回のみ
npx cap sync
npx cap open android     # Android Studio → Build → Build APK(s)
```

APK は `android/app/build/outputs/apk/` に出力されます。端末にコピー → 「提供元不明のアプリ」を許可 → タップしてインストール。Google サービスは不要です。

## 📖 使い方

1. アプリを開く → **⚙ 設定** → モデル(Qwen / DeepSeek)を選び、対応する **API キー**を貼る(端末内保存・アップロードなし)。
2. そのまま会話。数ターンで、あなたの「プロフィール」をバックグラウンドで抽出し始めます。
3. **キャラ** —— 設定 → コンパニオンキャラ で、内蔵テンプレから選ぶか自分で作成：名前・絵文字または画像アイコン・人格・専用ルール。上部バーでいつでも切り替え、各キャラが独立した会話を保持。**画面クリア**を押すと履歴は消さずに表示だけ消去。
4. **記憶** —— 覚えている内容を確認(個別削除可)。この記憶はすべてのキャラで共有されます。**記事を食べさせる**：リンクや本文を貼って興味の理解を形づくる。

> 🔑 API キーはご自身でご用意ください：[Alibaba DashScope(Qwen)](https://dashscope.console.aliyun.com/) または [DeepSeek](https://platform.deepseek.com/)。
