## ✨ 왜 AlterMe인가?

대부분의 AI 채팅 앱은 남의 서버에 얇은 껍데기를 씌운 것 —— 메시지와 키가 **그들의** 백엔드를 거칩니다. **AlterMe는 백엔드가 아예 없습니다.** 순수 온디바이스 WebView가 **당신의** 키로 모델 API에 직접 말을 겁니다. 당신에 대해 기억하는 모든 것은 폰 안에 남습니다.

게다가 작고 고치기 쉽습니다: JS 파일 약 6개, 프레임워크 없음, 빌드 과정 없음. 10분이면 "두뇌" 전체를 다 읽습니다.

## 🎯 주요 기능

- 🎭 **여러 캐릭터, 자유 전환** —— 페르소나가 하나뿐이 아닙니다: 파트너 / 들어주는 친구 / 놀이 친구 / 이성파 등 내장 템플릿을 자유롭게 추가·편집·삭제. 각 캐릭터가 고유한 성격·전용 규칙·대화 기록·유대 레벨을 가지며, 상단 바에서 원탭 전환하고 이미지 아바타도 설정 가능. "무슨 모델이야?"라고 물으면 여전히 솔직하게 답합니다(GPT인 척하지 않음).
- 🧠 **진짜 기억(캐릭터 간 공유)** —— 대화에서 당신의 장기 "프로필"을 조용히 추출하고, 모든 캐릭터가 이 "당신에 대한 이해"를 공유합니다. **글을 먹여**(링크나 본문 붙여넣기) 관심사 이해를 다듬을 수도 있습니다.
- 💞 **유대와 레벨(캐릭터별 독립)** —— 각 캐릭터와 따로 경험치가 쌓이고 각각 레벨업합니다. 작은 장치지만 의외로 다시 찾게 됩니다.
- 🔀 **두 모델, 원탭 전환** —— Qwen-Plus(알리바바)와 DeepSeek, 둘 다 OpenAI 호환. 언제든 전환.
- 🔒 **구조적 프라이버시** —— API 키는 로컬에만 저장. 우리 서버로는 아무것도 올라가지 않습니다(애초에 서버가 없음). 기억은 버튼 하나로 전부 삭제.
- 📱 **구글이 닿지 않는 곳에서도 작동** —— 순수 WebView, GMS 의존성 제로. Huawei / HarmonyOS 4.2에도 일반 APK로 깔끔하게 설치됩니다.
- 🪶 **개조를 전제로 한 설계** —— 순수 HTML/CSS/JS, 툴체인 불필요. `www/`만 열면 바로 수정 가능.

## 🚀 빠른 시작

### 먼저 브라우저에서 실행(개발 단계)

`www/`는 순수 정적 사이트입니다:

```bash
cd www
python3 -m http.server 8080
# http://localhost:8080 열기
```

> 데스크톱 브라우저에서는 모델 직접 호출이 CORS에 걸릴 수 있는데, 개발 단계에서만 나타나는 현상입니다. 폰에서는 CapacitorHttp를 거쳐 문제없이 동작합니다.

### APK 빌드(Android / HarmonyOS)

**Node + Android Studio + Android SDK**가 필요합니다.

```bash
npm install
npx cap add android      # 최초 1회
npx cap sync
npx cap open android     # Android Studio → Build → Build APK(s)
```

APK는 `android/app/build/outputs/apk/`에 생성됩니다. 폰으로 복사 → "출처를 알 수 없는 앱 설치" 허용 → 탭하여 설치. 구글 서비스 불필요.

## 📖 사용법

1. 앱 실행 → **⚙ 설정** → 모델(Qwen / DeepSeek) 선택 후 해당 **API 키** 붙여넣기(로컬 저장, 업로드 안 함).
2. 그냥 대화하세요. 몇 턴 지나면 백그라운드에서 당신의 "프로필"을 추출하기 시작합니다.
3. **캐릭터** —— 설정 → 동반자 캐릭터 에서 내장 템플릿을 고르거나 직접 만들기: 이름, 이모지 또는 이미지 아바타, 성격, 전용 규칙. 상단 바에서 언제든 전환하며 각자 독립된 대화를 유지. **화면 지우기**를 누르면 기록은 삭제하지 않고 화면만 비웁니다.
4. **기억** —— 무엇을 기억하는지 확인(개별 삭제 가능), 이 기억은 모든 캐릭터가 공유합니다. **글 먹이기**: 링크나 본문을 붙여 관심사 이해를 다듬기.

> 🔑 API 키는 직접 준비하셔야 합니다: [Alibaba DashScope(Qwen)](https://dashscope.console.aliyun.com/) 또는 [DeepSeek](https://platform.deepseek.com/).
